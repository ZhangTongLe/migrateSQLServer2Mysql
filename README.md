<h3>数据迁移工具(SQL server迁移到Mysql):</h3>
<ul>
	<li>(1).自定义选择待迁移表</li>
	<li>(2).自定义选择待迁移字段</li>
	<li>(3).目标数据库自动生成表(不存在时)</li>
	<li>(4).增加去重功能</li>
	<li>(5).增加导出.sql文件功能</li>
</ul>

<h3>该项目主要用来学习和巩固awt和swing的知识点:</h3>
<ul>
	<li>(1).布局(流布局,表格布局)</li>
	<li>(2).进度条(ProcessBar)</li>
</ul>

<h3>其他知识点:</h3>
<ul>
	<li>(1).JDBC</li>
	<li>(2).SQL语句(主要用到DDL,DML)</li>
	<li>(3).线程知识</li>
</ul>
